### Exercise 4.15
Give the values of `i` and `j` after each of the following expression statements
has been executed. (Assume that `i` has the value 1 initially and `j` has the
value 2.)

(a) `i += j;`  
(b) `i--;`  
(c) `i * j / i;`  
(d) `i % ++j;`

### Solution

(a) 3, 2  
(b) 0, 2  
(c) 1, 2  
(d) 1, 3
